<h1>Primeiros Passos</h1>

Antes de iniciar o desenvolvimento, foi necessário criar o ambiente virtual e instalar os pacotes necessários.

Foram feitos os passos abaixo:

1. Criar da pasta para armazenar o repositório do código e o material de apoio.

2. Criar a venv com o comando  `python -m venv venv`

3. Ativar a venv com o comando `.\venv\Scripts\activate.bat`

4. Criar o repositório remoto

5. Clonar o repositório remoto dentro da pasta criada anteriormente com o comando `git clone <link do repo>`

6. Entrar na pasta do repositório

7. Instalar bibliotecas necessárias com o comando `pip install <nomes das bibliotecas>`

8. Criar arquivo requirements.txt com o comando `pip freeze > requirements.txt`
